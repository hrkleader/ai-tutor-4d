Logické funkce
Logickou funkci jedné nezávisle proměnné lze vyjádřit čtyřmi způsoby (viz tabulka).
Vstupní proměnná je A a logické funkce jsou $f_0, f_1, f_2, f_3$;

| A | $f_0$ | $f_1$ | $f_2$ | $f_3$ |
|---|---|---|---|---|
| 0 | 0 | 0 | 1 | 1 |
| 1 | 0 | 1 | 0 | 1 |

$f_0$ – triviální identita, nulová funkce
$f_1$ – identická funkce, její hodnota se shoduje s hodnotou proměnné A
$f_2$ – funkce negace, hodnota funkce se shoduje s hodnotou $\overline{A}$
$f_3$ – triviální identita, jednotková funkce
Praktický význam mají pouze funkce $f_1$ a $f_2$. Negace se vyznačuje pruhem nad příslušnou proměnnou, např.
$f_2 = \overline{A}$ .

Logické funkce dvou proměných
Pro dvě vstupní proměnné A a B (n = 2) lze definovat celkem $2^{2^n}$ , tj 16 různých logických funkcí, označíme je $f_0$ až $f_{15}$, jejich přehled je uveden v následující tabulce:

| A | 0 | 0 | 1 | 1 | Význam jednotlivých funkcí |
|---|---|---|---|---|---|
| B | 0 | 1 | 0 | 1 | |
| $f_0$ | 0 | 0 | 0 | 0 | $f_0 = 0$; triviální identita |
| $f_1$ | 0 | 0 | 0 | 1 | $f_1 = A * B$; logický součin, AND |
| $f_2$ | 0 | 0 | 1 | 0 | $f_2 = A * \overline{B}$; přímá inhibice |
| $f_3$ | 0 | 0 | 1 | 1 | $f_3 = A$; funkce identická |
| $f_4$ | 0 | 1 | 0 | 0 | $f_4 = \overline{A} * B$; zpětná inhibice |
| $f_5$ | 0 | 1 | 0 | 1 | $f_5 = B$; funkce identická |
| $f_6$ | 0 | 1 | 1 | 0 | $f_6 = A \oplus B$; nonekvivalence, XOR, součet modulo 2 |
| $f_7$ | 0 | 1 | 1 | 1 | $f_7 = A + B$; logický součet, OR |
| $f_8$ | 1 | 0 | 0 | 0 | $f_8 = \overline{A + B}$; negovaný logický součet, NOR, Piercova funkce |
| $f_9$ | 1 | 0 | 0 | 1 | $f_9 = \overline{A \oplus B}$; ekvivalence, XNOR |
| $f_{10}$ | 1 | 0 | 1 | 0 | $f_{10} = \overline{B}$; negace |
| $f_{11}$ | 1 | 0 | 1 | 1 | $f_{11} = A + \overline{B}$; zpětná implikace |
| $f_{12}$ | 1 | 1 | 0 | 0 | $f_{12} = \overline{A}$; negace |
| $f_{13}$ | 1 | 1 | 0 | 1 | $f_{13} = \overline{A} + B$; přímá implikace |
| $f_{14}$ | 1 | 1 | 1 | 0 | $f_{14} = \overline{A * B}$; negovaný logický součin, NAND, Shefferova funkce |
| $f_{15}$ | 1 | 1 | 1 | 1 | $f_{15} = 1$; triviální identita |

V praxi se se používá malý počet základních členů
Základními funkcemi jsou:
Logický součin (funkce $f_1$) dvou proměnných A, B, označovaný jako AND.
Označíme-li výstup logického obvodu y, pak logický součin je zapsán $y = A * B$
Logický součet (funkce $f_7$) označovaný jako OR. Lze jej zapsat vztahem $y = A + B$
Negace (funkce $f_{10}$ a $f_{12}$) označovaná jako NOT. Tato funkce je dána vztahy $y = \overline{A}$ nebo $y = \overline{B}$ .
pomocí funkce NAND a NOR můžeme vyjádřit a realizovat jakoukoliv logickou funkci
nonekvivalence (funkce $f_6$) Funkce nabývá hodnoty 1, jsou−li hodnoty proměnných A a B různé.
Tato funkce je nazývána Exclusive OR a značí se XOR.
$y = A \oplus B = \overline{A} * B + A * \overline{B}$
ekvivalence (funkce $f_9$) funkce nabývá hodnoty 1, jsou−li hodnoty vstupních proměnných A a B stejné.
Tato funkce je negací předcházející funkce, značí se XNOR.
$y = \overline{A \oplus B} = A * B + \overline{A} * \overline{B}$
Funkce XOR a XNOR jsou významné pro realizaci aritmetických obvodů a číslicových komparátorů

Booleova algebra
Je to matematická disciplína
Používaná u při návrhu číslicových obvodů
základní operace
logický součin (konjunkce),
logický součet (disjunkce),
negace (inverze),
tvoří teoretický prostředník pro návrh logických obvodů s požadovaným chováním. Vztahy mezi dvouhodnotovými proměnnými Booleova algebra definuje svými zákony:

| | Zákony součtu | Zákony součinu |
|---|---|---|
| Zákon idempotence | $A + A = A$ | $A * A = A$ |
| Zákon absorbce | $A + A * B = A$ | $A * (A + B) = A$ |
| Zákon absorbce negace | $A + \overline{A} * B = A + B$ | $A * (\overline{A} + B) = A * B$ |
| Zákon komutativní | $A + B = B + A$ | $A * B = B * A$ |
| Zákon asociativní | $A + (B + C) = (A + B) + C$ | $A * (B * C) = (A * B) * C$ |
| Zákon distributivní | $A + (B * C) = (A + B) * (A + C)$ | $A * (B + C) = A * B + A * C$ |
| Neutrálnost nuly a jedničky | $A + 0 = A$ | $A * 1 = A$ |
| Agresívnost nuly a jedničky | $A + 1 = 1$ | $A * 0 = 0$ |
| Zákon vyloučeného třetího | $A + \overline{A} = 1$ | $A * \overline{A} = 0$ |
| Zákon negace | $\overline{0} = 1$ | $\overline{1} = 0$ |
| Zákon dvojité negace | $\overline{\overline{A}} = A$ | |
| De Morganovy zákony | $\overline{A + B} = \overline{A} * \overline{B}$ | $\overline{A * B} = \overline{A} + \overline{B}$ |

zákony Booleovy algebry jsou duální – jeden zákon vyplívá z druhého
lze zaměnit součin za součet a naopak při záměně logické 0 za 1
zákony Booleovy algebry jsou využívány při úpravách logických funkcí- zjednodušování funkcí
nazývané taky jako minimalizace logických funkcí
všechny logické součty zaměníme za logické součiny a naopak,
každou jednotlivou proměnnou negujeme,
negujeme celou rovnici.

Příklady:

Příklad 1: Upravte výraz $A + B * C + \overline{B} * \overline{C} + \overline{D}$ pomocí De Morganových zákonů.

Řešení:
$A + B * C + \overline{B} * \overline{C} + \overline{D} = \overline{\overline{A + B * C + \overline{B} * \overline{C} + \overline{D}}} = \overline{\overline{A} * \overline{B * C} * \overline{\overline{B} * \overline{C}} * \overline{\overline{D}}} =$
$= \overline{\overline{A} * \overline{B * C} * (B + \overline{C} * \overline{D})} = \overline{\overline{A} * (\overline{B} + \overline{C}) * (B + \overline{C} * \overline{D})} .$

Příklad 2: Upravte výraz $A + \overline{A} * B$ pomocí De Morganových zákonů.

Řešení:
$A + \overline{A} * B = \overline{\overline{A + \overline{A} * B}} = \overline{\overline{A} * \overline{\overline{A} * B}} = \overline{\overline{A} * (A + \overline{B})} = \overline{\overline{A} * A + \overline{A} * \overline{B}} = A + B .$

De Morganovy zákony umožňují realizaci logické funkce pomocí obvodů NAND nebo NOR.

Příklad 3: Logickou funkci $y = \overline{x}_1 * x_2 + x_1 * \overline{x}_3$ realizujte pomocí hradel typu NAND.

Řešení:
Zadanou logickou funkci upravíme s použitím De Morganova zákona. Aby nedošlo ke změně její hodnoty, provedeme dvojitou negaci.
$y = \overline{x}_1 * x_2 + x_1 * \overline{x}_3 = \overline{\overline{\overline{x}_1 * x_2 + x_1 * \overline{x}_3}} = \overline{\overline{\overline{x}_1 * x_2} * \overline{x_1 * \overline{x}_3}}$

Takto upravenou logickou funkci lze realizovat s použitím členů NAND (úpravou jsme převedli součet na součin) a na pravé straně rovnice jsou pouze součiny a negace. Realizace uvedené logické funkce vypadá následovně:
